# cafeteriaTech
Sistema de Pedidos para Cafeteria (Java)

Aplicação simples em Java para simular o atendimento de uma cafeteria via terminal. O sistema permite registrar pedidos, calcular valores com descontos, contabilizar doações e gerar um resumo final das vendas.

Funcionalidades
  Seleção de produtos por código
  Cálculo automático do valor total do pedido
  Desconto progressivo:
  10% para compras acima de R$150 e 5% para compras entre R$50 e R$150
  
  Promoção de cookies:
  
  Mais de 3 unidades: R$10 cada
  Até 3 unidades: R$12 cada
  Opção de doação de 2% do valor do pedido
  Registro de múltiplos pedidos
  
  Estatísticas finais:
  
  Total arrecadado
  Quantidade de pedidos
  Quantidade de capuccinos vendidos
  Média de doações
